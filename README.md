## Technical Insights

### Module 01: Memory & Data Types
In this module, I explored how C# manages data at the system level:
* **Integer (`int`)**: 4 bytes, range -2.1B to 2.1B. Ideal for general counters.
* **Long (`long`)**: 8 bytes. Essential for massive data and enterprise IDs.
* **Double (`double`)**: 8 bytes. Used for high-precision logistics calculations.
* **Boolean (`bool`)**: 1 byte. The foundation of logical branching.

### Module 02: Control Flow & Logic

* **User Input**: Handling interactions via `Console.ReadLine()`.
* **Decision Matrices**: Using `if/else` to assign transport based on cargo weight.
